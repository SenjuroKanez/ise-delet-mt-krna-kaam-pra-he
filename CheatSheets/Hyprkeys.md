# Hyprland Keybinds

## Main Modifier
- `SUPER`

---

# Applications

| Keybind | Action |
|---|---|
| `SUPER + Q` | Open terminal |
| `SUPER + E` | Open file manager |
| `SUPER + R` | Open app launcher |
| `SUPER + A` | Toggle launcher window |
| `SUPER + P` | Open power menu |
| `SUPER + L` | Lock screen (`hyprlock`) |

---

# Window Management

| Keybind | Action |
|---|---|
| `SUPER + C` | Kill active window |
| `SUPER + V` | Toggle floating |
| `SUPER + F` | Fullscreen |
| `SUPER + M` | Exit Hyprland |

---

# Focus Movement

| Keybind | Action |
|---|---|
| `SUPER + ←` | Focus left |
| `SUPER + →` | Focus right |
| `SUPER + ↑` | Focus up |
| `SUPER + ↓` | Focus down |

---

---

# Move Window To Workspace

| Keybind | Action |
|---|---|
| `SUPER + SHIFT + 1-0` | Move window to workspace 1-10 |

---

# Special Workspace

| Keybind | Action |
|---|---|
| `SUPER + S` | Toggle special workspace |
| `SUPER + SHIFT + S` | Move window to special workspace |

---

# Mouse Controls

| Keybind | Action |
|---|---|
| `SUPER + Mouse Left Drag` | Move window |
| `SUPER + Mouse Right Drag` | Resize window |
| `SUPER + Mouse Wheel Down` | Next workspace |
| `SUPER + Mouse Wheel Up` | Previous workspace |

---

# Screenshots

| Keybind | Action |
|---|---|
| `CTRL + SHIFT + T` | Save area screenshot |
| `SUPER + SHIFT + P` | Save full output screenshot |
| `SUPER + CTRL + S` | Copy area screenshot |

---
